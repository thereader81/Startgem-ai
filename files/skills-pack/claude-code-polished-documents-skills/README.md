# Polished Documents Skills Pack

This folder is a copyable skills bundle derived from `promptadvisers/claude-code-polished-documents-skills`.

## Included

All skills except the MCP-related builder (per request to skip Fire Crawl integration).

## How to Use

Copy this entire folder into any project. The skills live under:

```
.claude/skills/
```

You can also install into Codex using the skill-installer if preferred.
